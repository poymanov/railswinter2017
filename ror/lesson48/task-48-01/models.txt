Курс

Содержит информацию по курсам проекта. Здесь может храниться его название и какие-то другие уникальные данные.
Здесь имеются ввиду курсы "Настоящее программирование для всех 1", "Настоящее программирование для всех 2" и "Интенсив по Ruby On Rails"

УрокФайл

Хранит в себе информацию о приложенных к уроку файлах.
http://joxi.ru/Dr8dkQXI4v5Vdr

Хранит в себе название файла в структуре проекта и текстовое ("человеческое") описание.
Обычно все более-менее статические файлы (которые не будут часто перещаться, задача которых просто лежать и быть доступыми)
хранятся в какой-то определенной директории проекта, типа /files/static, поэтому нет смысла в базе хранить путь к файлу.
Просто имя и максимум поддиректорию, в которой этой файл может быть внутри основной директории.
Это спасает от того, что размещение проекта может поменяться и связь не будет потеряна.

| id | file | description | lesson_id |

ПользовательУрок

Хранит информацию о изученных пользователем уроках
Если записи в таблице нет, считаем, что пользователь не изучил урок. Имеется в виду вот это:
http://joxi.ru/Rmz9wd0F0wnX1r

При открытии пользователем страницы, мы проверяем наличие в этой таблице записи по id пользователя и id урока.
Если запись есть, выводим блок "Изучено", если нет то кнопку "Отметить изученным".
Эта кнопка как раз и добавляет в эту таблицу информацию о том, что такой-то пользователь просмотрел такой-то урок.

| id | user_id | lesson_id |
